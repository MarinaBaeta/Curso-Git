## Como baixar o projeto:

```bash
#Clonar o repositório
$ git clone https://github.com/MarinaBaeta/Curso-Git
#Entrar no diretório
$ cd git
#Instalar as dependências
$ yarn install
#Iniciar o projeto
$ yarn start
```
